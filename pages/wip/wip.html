Graphite

Graphite is an editor for writing interactive narrative games using a custom markup scripting language.

Features:

  • Natural-language syntax. The syntax of the Graphite language is intended to be easy to understand for writers with little to no programming experience.
  • Live playtesting. The game automatically refreshes and updates as you edit the script.
  • Error highlighting, as-you-type.
  • Multimedia integration. Simple integration of images and sounds.
  • Multiple export options. Export to JSON and HTML.
